## Interaction Flows

### Flow 1: Fill Required Field

```
1. User sees collapsed "Subject" box (red, large, required)
2. User clicks anywhere on box
3. Box expands with smooth animation (300ms)
4. Input field appears and auto-focuses
5. User types "A professional athlete"
6. AI suggestions appear below input
7. User continues typing or selects suggestion
8. User presses Escape or clicks outside
9. Box collapses with value preview visible
10. Check mark appears with bounce animation
11. Progress indicator updates (1/10)
```

### Flow 2: Switch Between Fields

```
1. User has "Subject" box expanded
2. User clicks "Action" box
3. "Subject" box collapses (300ms)
4. "Action" box expands (300ms) - simultaneous
5. Input in "Action" auto-focuses
6. User fills and collapses
7. Progress indicator updates (2/10)
```

### Flow 3: Complete and Continue

```
1. User fills "Subject" (required)
2. Progress: 1/10, Continue button disabled
3. User fills "Action" (required)
4. Progress: 2/10, Continue button ENABLED
5. Continue button glows with gradient
6. Keyboard hint appears: "Press Enter to continue"
7. User presses Enter or clicks Continue
8. Transitions to Summary Review step
```
